# JS Toolkit 🛠️

A premium, minimalist set of developer tools built with vanilla JavaScript, HTML5, and CSS3.

## Features
- **Text Utility**: Live word and character counting.
- **PassGen**: High-entropy password generator with custom length.
- **Color Hex**: Smooth RGB to HEX converter with live preview.

## Aesthetics
- **Glassmorphism Design**: Modern, translucent UI components.
- **Responsive Layout**: Designed for all screen sizes.
- **Animated Background**: Dynamic blobs for a premium feel.
- **Performance**: Zero dependencies, lightweight and fast.

## Setup
Simply open `index.html` in any modern web browser.
